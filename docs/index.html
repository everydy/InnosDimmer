Quick Start

InnosDimmer is built for a direct-HDMI INNOS 27QA100M secondary display on macOS. It dims by drawing click-through overlay windows and adjusts blue output through the macOS gamma table.

Launch the app

Open the Release build or local app bundle. The app appears in the macOS menu bar instead of as a normal Dock-first app.

Pick the target display

Use the app window or settings surface to confirm the selected external display. Keep the target as the INNOS display before testing dimming.

Adjust brightness and blue reduction

Brightness changes perceived screen brightness through the overlay. Blue reduction uses gamma adjustment and should restore the original table when cleared.

Use quick disable and restore

Quick disable returns to a bright working state. Restore previous dimming brings back the last software dimming values.

Default Shortcuts

The app does not intercept native brightness or media keys. It uses custom global shortcuts so macOS display controls remain untouched.

What The Status Means

Software dimming ready

No dimming command has been applied yet. This is the neutral launch state.

Overlay active

The app is changing perceived brightness through click-through overlay windows.

Platform blocked

macOS or the current surface prevented reliable dimming. Treat this as a limitation, not success.

Before You Rely On It

This app changes perceived brightness, not monitor backlight brightness. Full manual QA is still required for full-screen Spaces, presentation mode, protected playback, screen sharing, sleep/wake, HDMI reconnect, and shortcut behavior in different focused apps.
