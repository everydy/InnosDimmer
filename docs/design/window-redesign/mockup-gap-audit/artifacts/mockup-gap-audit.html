Defaults for implementation

Schedule row remove feature

Default: Fixed 3 rows. This preserves the current schedule model and avoids broad behavior changes.

Default: Matrix + log feed visible, raw text retained for export/copy support.

Default: Remove non-actionable explanatory copy. Keep operational labels and real status only.

Default: Add primitives first and keep controller in place until extraction becomes the smaller change.
